OPTION A - If the GitHub repo is EMPTY (recommended)
1) Extract this zip.
2) Open PowerShell.
3) Run these commands, changing the first cd path if you extracted somewhere else:

cd R:\Law_Order_Professional_Watch_Tracker_Package_ARTWORK_XML_FIXED\Law_Order_Professional_Tracker_Package

git init

git branch -M main

git remote add origin https://github.com/costaskk/Law-Order-Watch-Tracker.git

git add .

git commit -m "Initial Law Order watch tracker app"

git push -u origin main
