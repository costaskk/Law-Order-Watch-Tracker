Law & Order Watch Tracker - Visual Upgrade

What changed:
- Rebuilt the responsive layout so it uses desktop width correctly instead of looking cramped.
- Added improved show-specific artwork SVGs for every show.
- Added toast notifications for status changes, sync checks, exports, imports, and filter resets.
- Added confirmation modals for mark-next-watched, mark season watched/unwatched, and whole-show watched.
- Improved mobile spacing, buttons, episode cards, dashboard, progress, and contrast.
- Kept Vercel/GitHub Pages compatible static deployment.

To apply:
1. Extract this zip.
2. Copy everything into your local GitHub repo folder, replacing existing files.

   git pull --rebase origin main
   git push
