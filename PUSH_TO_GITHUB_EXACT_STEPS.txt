FIRST-TIME GITHUB UPLOAD - EXACT STEPS
Repo: https://github.com/costaskk/Law-Order-Watch-Tracker.git

1) Extract this ZIP somewhere easy, for example:
   R:\Law_Order_Professional_Tracker_Package_FIXED

2) Open PowerShell.

3) Go to the folder where you want to keep the GitHub repo:
   cd R:\

4) Clone your empty GitHub repo:
   git clone https://github.com/costaskk/Law-Order-Watch-Tracker.git

5) Enter the repo folder:
   cd R:\Law-Order-Watch-Tracker

6) Copy EVERYTHING from inside this package folder:
   Law_Order_Professional_Tracker_Package

   into this GitHub repo folder:
   R:\Law-Order-Watch-Tracker

   Important: do not copy the outer ZIP itself. Copy the contents: law_order_tracker_app, .github, scripts, Excel files, package.json, vercel.json, README files, etc.

7) Check files are there:
   dir

   You should see:
   law_order_tracker_app
   .github
   package.json
   vercel.json
   sync_trakt_and_excel.py
   README.txt

8) Stage all files:
   git add .

9) Commit:
   git commit -m "Initial Law Order watch tracker app"

10) Push:
   git push origin main

If push says branch main does not exist, run:
   git branch -M main
   git push -u origin main

If GitHub asks you to log in, use your GitHub username and a Personal Access Token, not your normal password.

GITHUB PAGES HOSTING
1) Open:
   https://github.com/costaskk/Law-Order-Watch-Tracker/settings/pages

2) Under Build and deployment:
   Source: Deploy from a branch
   Branch: main
   Folder: / root

3) Click Save.

4) Wait 1-3 minutes.

5) Open:
   https://costaskk.github.io/Law-Order-Watch-Tracker/law_order_tracker_app/

VERCEL HOSTING
1) Go to https://vercel.com
2) Add New Project
3) Import Git Repository
4) Choose costaskk/Law-Order-Watch-Tracker
5) Settings:
   Framework Preset: Other
   Build Command: leave empty
   Output Directory: law_order_tracker_app
   Install Command: leave empty
6) Deploy.

LOCAL TESTING - RECOMMENDED
Do not double-click index.html for testing. Run:
   start_local_app.bat

Or manually:
   cd R:\Law-Order-Watch-Tracker
   python -m http.server 8080

Then open:
   http://localhost:8080/law_order_tracker_app/
